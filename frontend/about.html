📖 About This Project

About ExoHabitAI

ExoHabitAI is an AI-powered web application developed as part of a Machine Learning project. It predicts whether an exoplanet could potentially support life, using a trained ML model and a rule-based weighted scoring algorithm.

🌌 The Problem

Why ExoHabitAI?

🔭
4,000+ Exoplanets

Scientists have discovered thousands of planets outside our solar system. Manually checking each for habitability is impossible.

🤖
AI Solution

Machine Learning analyzes multiple parameters simultaneously and gives instant predictions with confidence scores.

📊
Weighted Scoring

6 key parameters are weighted by scientific importance to generate accurate habitability scores.

🛠️ Development

How I Built It

This project was built step by step over multiple modules — from data preprocessing and model training to API development and frontend design.

Module 1–3: Data and ML Model
Collected NASA exoplanet data, preprocessed features, trained a Random Forest Classifier with 270 features including stellar spectral types.
Module 4: Model Evaluation
Evaluated model accuracy, selected best hyperparameters, saved final model as exohabit_model.pkl using Joblib.
Module 5: Backend API (Flask)
Built REST API with /predict and /rank endpoints, input validation, CORS configuration, and structured JSON responses.
Module 6: Frontend UI
Designed dark space-themed interface with Bootstrap 5, animated starfield, particle effects, real-time API connection, and prediction history.
